# Dáma - Tomáš Alexa

## Install

### Virtual environment

1. Create
    ```bash
    python3 -m venv .venv
    ```
1. Start environment
    ```bash
    source .venv/bin/activate
    ```
1. Install dependencies
    ```bash
    pip install -r requirements.txt
    ```

## Start the game

```bash
python main.py
```
